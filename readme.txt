== Description ==

Integrates W3Counter's free web stats service into your WordPress blog. This plugin adds a Blog Stats Dashboard to your admin area with real-time usage statistics for your blog. These include the number of visitors and page views you've received over the past 7 days and all time, top viewed posts, top referrers, top searches and top visitor locations. It also adds a sidebar widget for integrating W3Counter's tracking code into your theme.

Requires a free web stats account from <a href="http://www.w3counter.com">W3Counter.com</a>.

Want to donate to this plugin's development? Instead, upgrade your W3Counter account to the PRO plan.

== Installation ==

1. Upload `w3counter.php` to the `/wp-contents/plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to your admin dashboard and click the 'edit' link in the Blog Stats box to add your W3Counter website ID and password (provided at signup).
4. If you haven't already inserted the tracking code in your theme, go to the widgets page and drag the W3Counter Tracker widget onto your sidebar where you want the tracker to appear.
